# Fuck YT Shorts

A simple browser addon that hides YouTube Shorts recommendations by using CSS to hide the Shorts elements from the page.

## Features

- Removes Shorts shelf from YouTube homepage
- Hides Shorts suggestions from search results
- Lightweight and simple

## Installation

1. Clone or download this repository
2. Load the extension in your browser as an unpacked extension

## How it works

This addon injects CSS rules that hide YouTube Shorts elements from view.
